1. ゲーム概要

  • 「浦安めいすいくん総選挙」は浦安市のマスコットキャラクター「めいすいくん」たちに投票して仮想の浦安を1889年から2100年まで発展させる選挙体験ゲーム
  • タブレット端末（iPad等）3台から投票、大画面モニターでリアルタイムにマップが変化

7. エンディング・世界線

エンディングの流れ

  1. 2100年到達 → 白フラッシュ + 花火 + タイトルカード「2100年 到達！」（4秒）
  2. マップツアー（5箇所を巡回、各5秒）
  3. 世界タイプ発表（最多得票キャラで決定）
  4. 投票ランキング（この世界線の票数で表示）
  5. 選挙メッセージ「本物の選挙もあなたの一票が浦安を変える！」
  6. 「次の世界線へ」ボタン表示

世界線

「次の世界線へ」ボタンを押すと1889年に戻り新しいサイクル開始。今日の分の記録が街ランクに表示される。

10. ランダムイベント

  • ポジティブイベント（11種）: 5票ごとに発動チェック。おまつり、イチョウ並木、モルック大会など。
  • チャレンジイベント（6種）: たいふう、じしん、さかなが減るなど。

11. 運用ガイド

イベント当日の準備

  1. 大画面モニター: world.html をChromeで全画面表示（F11）
  2. タブレット端末（3台）: vote.html を開く
  3. 音楽開始: 画面クリックで音楽開始（ブラウザの自動再生制限のため）

管理操作

操作手順
日別得票数リセット街ランク → 「日別リセット」ボタン
全データリセット設定 → 「全データリセット」（復元不可）
デバッグ: +10年設定 → 「+10年進める」（3票追加）
デバッグ: +50年設定 → 「+50年進める」（15票追加）
スマホ操作設定 → 「スマホをしまう」/「電源を切る」

デプロイ

  • git add → git commit → git push でGitHub Pagesに自動デプロイ
  • BUILD_REV: world.html 内のBUILD_REV定数を変更のたびにインクリメント

12. 既知の課題（低優先度）

ID内容詳細
Bug3薄い票分布投票が全キャラに均等に分散した場合、バーグラフの差が小さくなり視覚的にわかりづらい。票数が少ないうちは特に目立つ。
Bug8同時投票時のoldLevel判定複数端末から同時に投票された場合、レベルアップ判定に使うoldLevelが正確に取得できず、レベルアップ演出が二重に発生したり、スキップされたりする可能性がある。
Bug92099年エッジケース最終ステップ（2091-2100年）付近で、エンディングトリガーのタイミングが世界線遷移と競合する可能性がある。ページロード後5秒間の誤発火防止ガードで軽減済み。
