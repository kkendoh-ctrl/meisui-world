# CLAUDE.md（めいすいワールド）

## プロジェクト概要
浦安市「うらやす市民まつり」（4/25-26）向けの投票シミュレーションゲーム。
来場者がiPadで投票すると、浦安市の地図がリアルタイムで変化する。

## 技術構成
- フロントエンド: React
- 地図描画: Mapbox GL JS
- バックエンド: AWS（API Gateway WebSocket + Lambda + DynamoDB）
- スケーリング: 1/50スケール（4,000人設定）

## 実装ルール

### コード規約
- コンポーネント1ファイルは200行以内。超える場合は分割する
- 状態管理はReact hooks（useState, useReducer）を使う
- API通信のエラーハンドリングは必ず入れる

### 地図・スケーリング
- 「小さな浦安」概念: 実人口の1/50で4,000人
- 投票率は実際の浦安市の投票率を自然に再現する設計
- 地図の変化は即座に全端末に反映（WebSocket経由）
